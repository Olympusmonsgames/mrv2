################
Teclas de Manejo
################

mrv2 viene con un poderoso sistema de teclas de manejo que permite cambiar o agregar a las que vienen por defecto.  Las teclas de manejo pueden ser cambiadas llengo a **Ventana->Teclas de Manejo**.

En la ventana de Teclas de Manejo, usted puede buscar la función o si una tecla está asignada.
