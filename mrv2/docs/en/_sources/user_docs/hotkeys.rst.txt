#######
Hotkeys
#######

mrv2 comes with a powerful hotkeys system which allows changing or adding to the default hotkeys.  The hotkeys can be changed by going to **Window->Hotkeys**.

On the Hotkeys window you can search the function or if a key is assigned to one.
