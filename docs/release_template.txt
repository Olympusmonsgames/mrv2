mrv2 v1.2.7
========

mrv2 is an open source professional player and review tool for vfx, animation and computer graphics for Windows, Linux and macOS.

This is a minor bug fix and feature update. 

Added hotkeys display to all UI buttons in the tooltips.
Better German, Portuguese, Chinese, French, Italian and Hindi translations.
Dramatically improved the performance of YUV420_P12 movie files like Netflix's Sol Levante demo clip.
Fixed use of nuke-default OCIO config from the command-line.
Work-around of DPX files that report incorrectly their FPS.
Better internationalization support showing commas for floating point numbers.
Wayland menu fix and a crash fix on start-up.
