mrv2 v1.1.9
=========

mrv2 is an open source professional player and review tool for vfx, animation and computer graphics for Windows, Linux and macOS.

This is a bug fix and feature update.  Several python improvements to both running scripts command-line and in the plug-in system.  Fixes to drag and drop now accepting http:// and similar protocols.  Fixes to LUTs.

Many Wayland fixes and improvments!
