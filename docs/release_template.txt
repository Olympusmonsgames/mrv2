mrv2 v1.3.6
=======

mrv2 is an open source professional player and review tool for vfx, animation and computer graphics for Windows, Linux and macOS.

This version improves on NDI, fixes a number of start up issues on Windows and improves the build process.


Main Web page (with links to videos and online documentation):
https://mrv2.sourceforge.io

Release History and Binaries:
https://github.com/ggarra13/mrv2/releases/
https://sourceforge.net/projects/mrv2/files/

Source code and build instructions:
https://www.github.com/ggarra13/mrv2
