mrv2 v1.1.0
========

mrv2 is an open source professional player and review tool for vfx, animation and computer graphics for Windows, Linux and macOS.

Numerous improvements to NDI and Windows installer. Also a huge Python Panel bug fix and saving format fixes for TIFF and PNG.  Smaller footprint of installers and installed size and quicker packaging for those compiling from source.
